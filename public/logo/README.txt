Drop your logo files here.

Recommended filename for the header logo: logo.png
If you want to use your file `1Asset 1@1x.png`, copy or rename it to `logo.png` in this folder.

URL to use in your app: /logo/logo.png

Example <img> tag: <img src="/logo/logo.png" alt="Site logo" className="h-10 w-10" />

If you prefer to keep the original filename, reference it directly, e.g. /logo/1Asset%201@1x.png (space must be encoded as %20).
